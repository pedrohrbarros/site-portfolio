## Getting Started <a name = "getting_started"></a>

Clone the project with:

```
git clone https://github.com/pedrohrbarros/site-portfolio.git
```

Move to directory

```
cd site-portfolio
```

Install al the dependecies

```
npm install --noptional
```

Run the project locally

```
npm run dev
```

### Prerequisites

- Node.js
- Next.js
- Typescript
- TailwindCSS
